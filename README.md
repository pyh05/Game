# 게임 정보 관리 홈페이지
- 게임 정보를 입력하여 그 게임에 관한 정보를 간단하게 볼 수 있다

# 홈페이지 설명
- 게임 정보
- 게임 등록
- 게임 목록
- 게임 정보 수정/삭제
